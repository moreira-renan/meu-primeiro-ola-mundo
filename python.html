Python



Hoje a linguagem de programação Python é uma das linguagens mais populares, por ser utilizada em vários setores como jogos, páginas de internet, programas, entre outros, e por ser uma linguagem de mais simples compreensão. O Python foi criado em Amsterdã capital da Holanda em 1989 pelo programador Guido Van Rossum, inspirado pela linguagem ABC que ele estava trabalhando na época. Outra curiosidade é que o nome Python não é por conta da cobra de mesmo nome, conhecida no Brasil de píton, o nome foi uma homenagem a um programa de TV chamado de Monty Python Flying Circus de 1969, mas mesmo assim a cobra foi adotada como mascote da linguagem como mostrado na figura a seguir:

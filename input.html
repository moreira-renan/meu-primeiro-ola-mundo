Função input()



Com o Python podemos solicitar que o próprio usuário insira o valor dentro da variável pelo comando input, com ele não precisamos pré-definir os valores das variáveis. Vamos praticar com um exemplo:

idade = input('Qual sua idade')

print(idade)

Resultado:



Para realizar calculo com os valores inseridos pela função input() teremos que mostrar ao Python que o valor inserido se trata de um número e não um texto, para isto utilizaremos a função int(), veja a seguir:

nota1 = int(input('Qual a primeira nota?'))

nota2 = int(input('Qual a segunda nota?'))

resultado = (nota1 + nota2)

print(resultado)

Lógica de programação



O que é lógica? Você parou para pensar que tudo que fazemos tem uma sequência, uma ordem? Ao acordar você se veste, escova os dentes, toma café da manhã entre outras atividades. Todas estas atividades tem uma lógica, por exemplo, por que se vestir pela manhã? Imagina se todos fossem de pijama para a escola ou trabalho? Para cada tarefa temos uma resposta racional para ela, de maneira semelhante pode se dizer que a lógica de programação é a maneira que instruções são organizadas em um algoritmo para o funcionamento de um programa. Algoritmo???? O algoritmo nada mais é que a sequência de passos a ser realizado para que uma tarefa seja feita. Vamos praticar com um exemplo de como comer um pão:



  • Pegue o pão dentro da embalagem.
  • Com uma faca parta o pão ao meio.
  • Com a ponta da faca passe na manteiga.
  • Passe a manteiga no pão.
  • Coma o pão.
  • Fim.


Simples não é mesmo? Porém como falamos anteriormente o computador não sabe falar nossa língua, então devemos utilizar alguma das linguagens de programação, como o Python para criarmos algoritmos e por fim programas. Na sequência vamos falar um pouco mais sobre o Python.

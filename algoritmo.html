Existem vários comandos e vários códigos para diversas tarefas, porém como falado o objetivo do curso não é nos tornamos programadores, mas darmos início nossa jornada. Podemos até fazer cálculos utilizando o Python, vamos praticar? Clique no botão “+ Código” e digite o comando:

print(2 + 5)

O resultado do cálculo foi 7, mas por que não colocamos os números entre aspas? Vamos testar esta alternativa para verificarmos a diferença, digite o comando:

print(‘2 + 5’)

Desta vez o resultado foi 2 + 5, quando digitamos letras e número entre parentes o Python nos mostra exatamente o texto digitado, já o comando sem aspas entende-se que queremos realizar o cálculo. Podemos realizar não somente operação de adição, mas de qualquer outra operação, basta substituir o símbolo “+” por algum dos símbolos a seguir:


  • + → adição
  • - → subtração
  • * → multiplicação
  • / → divisão
  • ** → potenciação

Exemplos:
print(50 / 2) → resultado = 25
print(20 * 30) → resultado = 600
print(100 - 2) → resultado = 98
print(5 ** 2) → resultado = 25

Você pode também utilizar o parêntese para definir a ordem do cálculo (operação entre parênteses será calculada primeiro, na sequência será calculada a operação de fora dos parênteses), como por exemplo a média entre 4, 5 e 6:

print((4+5+6)/3)

De mesma forma, não colocamos entre aspas o valor numérico a ser calculado, para que o Python entenda que o conteúdo da variável será um número que poderá ser calculado e não apenas um texto.

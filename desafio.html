Desafio final!



Let’s go!! Nosso desafio final consiste em criar um algoritmo para calcular a média escolar de um aluno, com os conteúdos vistos até este momento fica fácil não é mesmo?
Dicas: faça um algoritmo que receba 6 variáveis, nome do aluno, matéria, nota do primeiro bimestre, nota do segundo bimestre, nota do terceiro bimestre e nota do quarto bimestre (lembre-se de colocar a função para indicar que o valor se trata de um número inteiro, ex: nota = int(input('Qual a nota?')) ), após este input criar uma variável que fará o algoritmo fazer o cálculo da média (caso tenha dúvida sobre como calcular a média fale com seu professor de matemática), após o cálculo fazer uma nova variável utilizando a função format() com a frase “A média do aluno x na matéria y foi de z.”
